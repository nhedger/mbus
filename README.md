# M-Bus

This repository aims to provide a comprehensive **Rust implementation of the
M-Bus (Meter-Bus) protocol**.

## Crates Overview

This workspace contains the following crates:

- ✅ `mbus-frame`: M-Bus link-layer frames structures and serialization logic.
- ⚠️ `mbus-frame-parser`: M-Bus frame parser (work in progress).
